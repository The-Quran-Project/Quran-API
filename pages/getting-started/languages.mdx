## Available Languages

Currently, 4 major translations are given with **every** response.
- Arabic _(with and without tashkeel)_
- English
- Bengali
- Urdu

Other non-major translations, that are **not** given with every response, but can be found in the [Get Full Translation Page](/getting-started/get-full-quran-translation) are:
- turkish
- uzbek


In request and responses, this is how the languages are used:

- `english` - English Translation
- `arabic1` - Arabic Translation with Tashkeel (diacritics)
- `arabic2` - Arabic Translation without Tashkeel (diacritics)
- `bengali` - Bengali Translation
- `urdu` - Urdu Translation
- `turkish` - Trukish Translation
- `uzbek` - Uzbek Translation


We will be adding more translations in future. If you want your language's translation to be added, [contact us](/contact) and we'll add it as soon as we can.
